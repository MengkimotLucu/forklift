# Folder Gambar Satria Mandiri Forklift

## Struktur Folder yang Dibutuhkan:

Buatlah struktur folder berikut di dalam `assets/images/`:

```
assets/images/
├── gallery/
│   ├── forklift-7ton-cimareme.jpg
│   ├── forklift-5ton-gedebage.jpg
│   ├── forklift-3ton-kopo.jpg
│   ├── forklift-7ton-batujajar.jpg
│   ├── forklift-5ton-padalarang.jpg
│   └── forklift-3ton-cimahi.jpg
└── hero_forklift_7ton.png (sudah ada)
```

## Cara Membuat Folder:

### Windows:
1. Buka File Explorer
2. Navigasi ke `C:\Users\LENOVO\Documents\Forklift\assets\`
3. Buat folder baru bernama `images`
4. Di dalam folder `images`, buat folder baru bernama `gallery`

### Atau via Command Prompt:
```cmd
cd C:\Users\LENOVO\Documents\Forklift\assets
mkdir images
cd images
mkdir gallery
```

## Spesifikasi Gambar:

- **Format:** JPG atau PNG
- **Ukuran:** Minimal 800x600 pixel
- **Kualitas:** Medium (70-80%) untuk loading cepat
- **Naming:** Gunakan nama file yang sesuai dengan SEO (contoh: `rental-forklift-7-ton-bandung.jpg`)

## Placeholder Images:

## SEO Tips untuk Gambar:

1. **Alt Text:** Sudah di-set di HTML dengan keyword yang relevan
2. **File Size:** Kompres gambar sebelum upload (maks 200KB per gambar)
3. **File Name:** Gunakan nama deskriptif dengan keyword
4. **Format:** Gunakan WebP jika browser mendukung (lebih ringan)
